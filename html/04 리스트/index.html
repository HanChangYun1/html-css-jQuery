목록 출력

  • ul(unorded list) type속성 : disc, circle, square
  • ol(orded list) type속성 : 1, a, A, i, I
  • dl(definition list) : 용어정리를 의미, dt, dd를 자식으로 가져야 함

ul(순서가 중요하지 않는 목록)

예시:

올해 개봉작

  • 플래시
  • 가디언즈..
  • 범죄도시

ol (순서가 존재하는 목록)

예시

올해 영화 흥행순위

  1. 가디언즈..
  2. 범죄도시
  3. 플래시

dl (용어의 의미)

예시

클라이언트 사이드 랭귀지

html
하이퍼 텍스트 마크업 랭귀지
css
캐스캐이딩 스타일 시트
