display level

display의 종류

block 레벨

  • 한줄을 통째로 차지하는 요소
  • 기본적으로 부모의 넓이와 동일한 넓이를 가지고있음
  • 크기를 가질 수 있음

inline 레벨

  • 줄내림이 되지 않는 요소
  • 크기속성을 가질 수 없음

inline block 레벨

  • 줄내림이 되지 않으면서, 크기를 가질 수 있음
  • img, svg, canvas, video, iframe, button, input, select, textarea

inline 레벨간의 수평 줄맞춤

span span span
  • text-align은 요소 안쪽 글자 및 인라인계열요소를 정렬
  • line-height은 행간을 제어하는 속성이지만, 요소의 높이와 동일하게 설정할 경우 수직 중앙정렬기능으로 사용된다 (!한줄일 경우만 사용)
  • vertical-align속성은 글자 및 일라인계열요소의 수평줄 맞춤 기능으로 사용

block레벨의 수평정렬

  • margin의 수평값에 auto값을 사용(auto값은 block레벨에만 적용됨)
