grid layout

  • grid속성은 block레벨과 유사, 노드순서와 상관없이 layout을 구현할 수 있음
  • grid분할예시 : repeat(3, auto), repeat(3, 1fr), repeat(3,100px)
  • grid요소의 자식은 크기를 지정하지 않을시 최소 그리드 한개의 크기를 갖는다
  • 자식요소의 grid칸 설정예시 :
    grid-column: 2/4;
    grid-column: 2/span2(2~3칸 소유)
    grid-colums: 3(3칸에서 시작)
    grid-area : 줄시작/ 칸시작/ 줄끝/ 칸끝
